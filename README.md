# Blog App

A responsive blog application built using Next.js and Tailwind CSS. It allows users to browse blog posts and view detailed content.

## Features

* Responsive layout for all screen sizes
* Blog listing with pagination
* Dynamic blog detail pages
* Clean and modern UI

## Tech Stack

* Next.js
* Tailwind CSS
* JavaScript
